## 版本概述

### 版本目标

本版本是 Subhuti AI Agent 框架的初始发布版本，目标是建立完整的工程化基础设施，包括：

- 核心 AI Agent 框架（LLM + Tool + Memory）
- HTTP 服务网关
- Docker 容器化支持
- 完整的工程化工具链（Makefile、pre-commit、配置系统）
- 自动化发布验证流程

### 范围

**包含**：
- ✅ 核心 Agent 框架
- ✅ HTTP Server（Axum）
- ✅ 心理咨询专家插件
- ✅ TOML 配置系统
- ✅ Docker 构建和部署
- ✅ 工程化工具链
- ✅ 发布验证流程

**不包含**：
- ❌ 多专家并发支持（v0.2.0）
- ❌ 插件市场（v0.3.0）
- ❌ Web UI 管理界面（v0.4.0）

### 依赖关系

- 前置版本：无（初始版本）
- 外部依赖：
  - Doubao LLM API
  - PostgreSQL 14+ with pgvector
